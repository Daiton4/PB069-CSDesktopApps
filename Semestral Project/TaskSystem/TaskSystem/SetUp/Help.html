Task System Manual and Help

Set Up

Task System works best with Microsoft SQLServer databases. This version is prepared for use with localDb. After Instalation of your version of MS SQL server, get information on avaiable servers in command line by command "SQLLocalDB info". Choose your instance and use "SQLLocalDB start YourInstanceName". Then get connection link with "SQLLocalDB info YourInstance". Dont close command line.

Start Microsoft SQL Management Studio. Open file StartupQuery.sql located in folder TSFront/SetUp or TaskSystem/SetUp in management studio. In top section of file change properties FILENAME to desired file location (values given wont work on your computer). After this step your DB is created and you can close management studio.

In SetUp folder (TSFront for GUI, TaskSystem for console) open settings file and set value after "Database Link^" to your database link (same as in management studio login).
Now you can login by name and password both "testuser"

Using Application GUI

  1. Login Screen

    2. On login screen you are asked to provide your credentials, username you were given (case insensitive) and password. In case you enter wrong information, message in red will appear, otherwise you will proceed to home section.

  2. Home Screen

    3. All controls are accesible through home screen. By default you can see here all your tasks and total time it takes to finish them. When you click on task, its prerequisities will show up. You can edit prerequisities by selecting task and pressing Edit over Prerequisities window.
